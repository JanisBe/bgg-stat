# BggStat

Projekt do wizualizowania statystyk z boardgamegeeka.
Strona dostępna tutaj [https://janisbe.github.io/bgg-stat/](https://janisbe.github.io/bgg-stat/) 
Można wgrać swoje csv wyeksportowane z bgg albo posłużyć się moim jako przykładem.
Opcja eksportu z bgg jest dostępna na stronie, trzeba być zalogowanym w bgg.
